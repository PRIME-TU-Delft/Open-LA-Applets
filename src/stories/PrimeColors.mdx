 ## Colors

The colors are adapted from the [TU Delft Prime colors](https://www.tudelft.nl/huisstijl/bouwstenen/kleur).
When you call `PrimeColor.getRandomColor()`, it will return a random color from the palette.
Calling `PrimeColors.asArray()` will return an array of all the colors in the following order:

### Opacity

To set the opacity of a color, you can use the `opacity` function. This function takes a number between 0 and 1 and returns a string with the opacity value.

```ts
PrimeColor.opacity(0.5); // "7F"
```

This value can be appended to the end of a color string to set the opacity. For example:

```ts
PrimeColor.blue + PrimeColor.opacity(0.5); // "#0076C27F"
```
